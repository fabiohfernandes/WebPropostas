## ğŸ”’ Technical Architecture Summary

### Frontend Stack
- **Framework:** Next.js 14 with App Router
- **Language:** TypeScript 5.3+
- **State:** Zustand (global) + React Context (builder)
- **Drag-Drop:** dnd-kit (performance-optimized)
- **Canvas:** Fabric.js or Konva.js (element manipulation)
- **Rich Text:** TipTap (structured editing)
- **Charts:** Recharts (data visualization)

### Backend APIs
- `POST /api/v1/templates/create` - Create template
- `POST /api/v1/ai/generate-text` - GPT-4 text generation
- `POST /api/v1/ai/generate-image` - DALL-E 3 image generation
- `POST /api/v1/assets/upload` - Image/video upload
- `GET /api/v1/templates/my-library` - User's templates (Professional)

### Database Extensions
```sql
-- Templates (existing table)
templates (id, organization_id, name, category, sector, fields, content_template)

-- User Template Library (Professional tier)
user_templates (id, user_id, template_id, is_favorite, last_used_at)

-- Builder Sessions (auto-save)
builder_sessions (id, user_id, template_id, canvas_state, last_saved_at)

-- AI Usage Tracking (token limits)
ai_usage_logs (id, user_id, operation_type, tokens_used, cost_usd, created_at)
```

### OpenAI Integration
- **Text AI:** GPT-4 Turbo ($0.01 per 1K tokens)
- **Image AI:** DALL-E 3 ($0.04 per image)
- **Token Limits:** 0 (Freemium) / 50K (Standard) / 200K (Professional)
