### `/ai-builder/` - AI Proposal Builder (Week 1 Project)
**Date:** October 2, 2025
**Status:** Completed as standalone project

A separate AI-powered proposal builder project that was completed in Week 1. This was an experimental project focused on AI proposal generation using OpenAI GPT-4o/o1.

**What it was:**
- 4-step wizard for proposal generation
- OpenAI GPT-4o (fast) and GPT-o1 (deep reasoning) integration
- Split-view editor with preview and chat
- AI chat for proposal refinements
- Version tracking system

**Why archived:**
- Completed as standalone project (not integrated into main platform)
- Different approach from main WebPropostas platform
- Kept for reference and potential future integration
